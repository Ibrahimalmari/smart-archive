# Smart Archive System

Smart e-Archive System — Laravel + AI + OCR hybrid document management system with approval workflow, intelligent search, and metadata extraction.

---

## 🚀 Features

- 📄 Document Management System
- 🧠 AI-powered classification (coming soon)
- 🔍 Intelligent search
- 🏢 Organizations & Departments structure
- 👥 User roles & permissions (Sanctum)
- 🔐 Secure API authentication
- 📤 File upload & storage system
- ✅ Approval workflow system

---

## 🛠️ Tech Stack

- Laravel (Backend)
- Laravel Sanctum (Auth)
- MySQL / PostgreSQL
- REST API Architecture
- Clean Architecture (Services, Repositories, DTOs)

---

## 📦 Project Status

🚧 Currently in development  
✔️ Core system completed (users, orgs, departments)  
⏳ AI & OCR integration coming next

---

## ⚙️ Installation

```bash
git clone https://github.com/Ibrahimalmari/smart-archive.git
cd smart-archive
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
